Literal "pet" project

This is my personal project using an ESP32 microcontroller and a temperature sensor to monitor our second appartment gas heating system. When the temperature drops too low, the ESP32 sends a warning through a Telegram bot so my father knows the heating has turned off and can prevent the pipes from freezing.

The ESP32 also drives a small TFT display where I can show sensor data but most importantly... pictures of my CAT!
